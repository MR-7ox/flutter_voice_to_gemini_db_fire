# flutter_voice_to_gemini_db_fire

A Flutter application that captures voice input, sends it to Google Gemini for processing/analysis, and stores the results in Firebase.

Features  
- Real-time speech recognition from microphone input  
- Integration with Gemini (Google’s AI) to analyse/process voice data  
- Storage of processed results in Firebase—Firestore / Realtime DB (configurable)  
- Cross-platform support for mobile (iOS & Android) and optionally web/desktop
